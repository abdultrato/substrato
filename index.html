Arquitetura

Runtime modular com contratos gerados

O backend concentra regras de domínio em apps Django e expõe API versionada. O frontend Next.js consome o contrato OpenAPI, aplica RBAC e usa páginas geradas para acelerar CRUDs sem perder formulários específicos quando o fluxo exige comportamento próprio.
