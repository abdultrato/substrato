🚀 COMO USAR PARA GERAR 50+ PÁGINAS DE CRUD
═════════════════════════════════════════════

Passo 1: ABRA OS ARQUIVOS GERADOS
   → SUBSTRATO_MAPEAMENTO_COMPLETO.json (estruturado)
   → MAPEAMENTO_VISUAL.md (visual)

Passo 2: ESCOLHA UM MODELO
   Exemplos prontos:
   • clinical/Patient (57 campos)
   • billing/Invoice (25+ campos)
   • pharmacy/Product (30+ campos)
   • nursing/Procedure (20+ campos)

Passo 3: COPIE O PADRÃO DO AUTOFORM
   
   Para CREATE:
   ┌─────────────────────────────────────────────────┐
   │ <AutoForm                                       │
   │   endpoint="/api/v1/clinical/patient/"          │
   │   method="post"                                 │
   │   submitLabel="Criar Paciente"                  │
   │   onSuccess={(data) => {                        │
   │     router.push(`/patient/${data.id}`)          │
   │   }}                                            │
   │   config={{                                     │
   │     etapas: [                                   │
   │       { titulo: "Dados Pessoais",               │
   │         campos: ["name", "birth_date", ...] }, │
   │       { titulo: "Endereço",                     │
   │         campos: ["address_street", ...] }      │
   │     ]                                           │
   │   }}                                            │
   │ />                                              │
   └─────────────────────────────────────────────────┘

   Para UPDATE:
   ┌─────────────────────────────────────────────────┐
   │ <AutoForm                                       │
   │   endpoint="/api/v1/clinical/patient/"          │
   │   method="put"                                  │
   │   initialValues={pacient}                       │
   │   config={{                                     │
   │     somenteLeituraCampos: ["id", "created_at"] │
   │   }}                                            │
   │ />                                              │
   └─────────────────────────────────────────────────┘

Passo 4: CUSTOMIZE COM SEUS CAMPOS
   Todos os campos estão em SUBSTRATO_MAPEAMENTO_COMPLETO.json
   Copie os campos_principais de cada modelo

Passo 5: REPITA PARA CADA MODELO/PÁGINA
   290 endpoints × 2-3 páginas por endpoint = 50+ páginas automático!
